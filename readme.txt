Structure du projet :
		DOSSIER			CONTENU
		-executables	-contient l'exectuable de MustacheBoxCalculator, notre calculateur de valeurs pour la boîte à moustache
		-data			-contient les données de nos metriques nos implementations et autres outils
			-tache1		-contient le fichier CSV des valeurs pour les boîtes à moustache calculées avec MustacheBoxCalculator
			-tache2		-contient les fichiers CSV des données avec et sans filtrage des points extrêmes pour les corrélations entre TLOC 				et TASSERT, et entre WMC et TASSERT
			-tache3		-contient le fichier .xlsx Excel contenant les données et les graphiques utiles pour la quasi-expérience
		-src			-contient le code source de l'implémentation de MustacheBoxCalculator

		FICHIER			CONTENU
		rapport.pdf		-le rapport du projet
